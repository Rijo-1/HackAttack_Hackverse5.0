## 🚀 Getting Started

### Prerequisites
bash
node >= 16.0.0
npm >= 8.0.0


### Installation
1. Clone the repository
bash
git clone https://github.com/yourusername/GyannDCX.git


2. Install dependencies
bash
cd GyannDCX
npm install


3. Set up environment variables
bash
cp .env.example .env
# Edit .env with your configuration


4. Start the development server
bash
npm run dev
